### 簡介
本專案是基於 Kubernetes、Node.js 和 MySQL 的電子商務網站，旨在展示微服務架構的設計與實現，以及 CI/CD 流程。專案包含多個微服務，涵蓋使用者管理、產品管理、訂單處理和付款服務。透過 Jenkins 自動化工具，使用 Terraform 進行基礎架構的配置，並使用 Docker、Helm 和 Kubernetes 進行容器化與部署。

目前開發中。
